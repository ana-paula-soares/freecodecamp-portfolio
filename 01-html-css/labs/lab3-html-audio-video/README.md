The purpose of this lab is to practice integrating multimedia elements into HTML documents, specifically focusing on:

* **Video embedding** (`<video>`) for displaying visual content
* **Audio embedding** (`<audio>`) for playing sound files
* **Media Sources** (`<source>`) or attributes (`src`) to define file paths
* **Media Attributes** (`controls`, `loop`) to manage playback behavior
